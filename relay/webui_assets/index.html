What routing settings mean
Privacy mode
Controls when sensitive subtasks must stay on your local model instead of cloud.
Balanced
High-sensitivity tasks and prompts mentioning secrets stay local. Difficult work still routes to cloud when enabled.
Strict local
Also keeps medium-sensitivity subtasks local. Use when you want tighter data control.
Permissive cloud
Does not force local for sensitivity alone — only explicit secret keywords still stay local.
Confidence
For simple local tasks only: if the local answer scores below this (0–1), Relay retries on cloud.
Max subtasks
Cap on subtasks per request. Independent subtasks run in parallel up to this limit.
Auto routing
Relay picks local vs cloud from capabilities. Difficult work (coding, math, proofs) never runs locally when cloud is on.
